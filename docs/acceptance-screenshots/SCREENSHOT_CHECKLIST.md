# 验收截图清单

本轮实机验证使用临时 Tomcat 访问路径：

`http://127.0.0.1:18090/housingfund-collection/`

浏览器自动截图工具在本机被企业策略禁止访问 `localhost/127.0.0.1`，因此未能自动落盘截图。验收前请按以下清单手工截图，页面状态与本轮 HTTP 表单和数据库验证结果一致。

1. 首页或导航页
   - URL：`/`
   - 截图要点：显示“住房公积金管理系统”标题和 7 个业务入口。

2. 系统参数维护页
   - URL：`/params?seqname=ACCNUM`
   - 截图要点：查询结果包含 `UNITACCNUM`、`PERACCNUM`；普通参数可新增、修改、删除；账号序列参数删除受保护。

3. 单位开户成功页
   - URL：`/units/open`
   - 截图要点：单位开户回单，测试单位账号 `000000900100`，状态正常，开户日期显示。

4. 个人手工开户成功页
   - URL：`/persons/open`
   - 截图要点：个人开户回单，个人账号 `000000900100`，单位人数和汇总金额同步。

5. 个人 Excel 批量导入结果页
   - URL：`/persons/open`
   - 截图要点：失败批次显示 `成功 0 条` 和失败原因；成功批次显示 `成功 2 条`、`失败 0 条`。

6. 单位资料修改成功页
   - URL：`/units/edit`
   - 截图要点：单位资料变更回单；销户单位查询时提示不可修改。

7. 个人资料普通修改成功页
   - URL：`/persons/edit`
   - 截图要点：个人资料变更回单，测试个人姓名改为“验收个人改”。

8. 个人资料强制变更成功页
   - URL：`/persons/edit`
   - 截图要点：冲突确认页显示占用账户 `000000900003`；强制变更回单显示新建错误账户 `000000900103`。

9. 单位信息查询结果页
   - URL：`/units/query`

10. 个人信息查询结果页

12. Tomcat 启动成功日志

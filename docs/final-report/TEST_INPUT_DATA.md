# 页面测试输入数据

本文档用于手工测试住房公积金管理系统——筹集子系统各页面。建议先使用测试库重新导入：

```text
schema.sql -> data.sql -> demo-data.sql
```

如果没有重置数据库，账号序列和部分证件号码可能已经被使用。遇到“组织机构代码重复”“证件号码已开户”时，优先换用本文档提供的备用数据。

- 访问路径：`http://127.0.0.1:18090/housingfund-collection/`
- 推荐测试库：`housingfund_collection_acceptance`

## 11. 数据库核对 SQL

### 11.1 查询账号序列

```sql
SELECT SEQNAME, SEQ, MAXSEQ, `DESC`, FREEUSE1
FROM TB001
WHERE SEQNAME IN ('UNITACCNUM', 'PERACCNUM');
```

### 11.2 查询单位

```sql
SELECT UNITACCNUM, UNITACCNAME, ORGCODE, ACCSTATE, PERSNUM,
       BASENUMBER, UNITPAYSUM, PERPAYSUM, UNITPROP, PERPROP
FROM TB002
WHERE UNITACCNUM IN ('000000900001', '000000900002', '000000900099')
   OR UNITACCNAME LIKE '%页面测试%'
   OR UNITACCNAME LIKE '%演示资料修改%'
ORDER BY UNITACCNUM;
```

### 11.3 查询个人

```sql
SELECT ACCNUM, UNITACCNUM, PERNAME, IDTYPE, IDCARD, PERACCSTATE,
       BASENUMBER, UNITPROP, INDIPROP, UNITMONPAYSUM, PERMONPAYSUM
FROM TB003
WHERE ACCNUM IN ('000000900001', '000000900002', '000000900003', '000000900004')
   OR PERNAME LIKE '%页面%'
   OR PERNAME LIKE '%批量%'
   OR IDCARD IN (
       '110105199501010017',
       '110105199502020014',
       '110105199506060013',
       '110105199507070010',
       '110105199303030033',
       '910105199303030033'
   )
ORDER BY ACCNUM;
```

### 11.4 强制变更专项核对

```sql
SELECT ACCNUM, PERNAME, IDCARD, PERACCSTATE, REMARK
FROM TB003
WHERE ACCNUM IN ('000000900001', '000000900003')
   OR IDCARD IN ('110105199303030033', '910105199303030033')
ORDER BY ACCNUM;
```
